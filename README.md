##  About The Project
**Study Buddy** is a mobile application designed to help university students easily find study partners, form discussion groups, and organize study sessions. 

This repository contains the **High-Fidelity UI/UX Mockup** developed for the **ISB 26603 Mobile and Ubiquitous Computing** course at **UniKL MIIT**. The current build focuses on demonstrating the user journey, structural layout, and front-end navigation of the proposed application.

###  Core Objectives
* Provide a clean, intuitive interface for students to connect academically.
* Demonstrate the utilization of smart device features (Touch interaction, GPS/Location mapping integration).
* Establish a seamless user flow from authentication to scheduling.

---

##  Features & Screens

The application mockup consists of several interconnected screens demonstrating the complete user journey:

* **Authentication:** Login and Registration screens with a clean, two-tone curved UI.
* **Dashboard:** A scrollable grid view of available study groups and personal enrolled groups.
* **Group Management:** * Full-screen interface to create new public/private groups.
  * Upload group photos and set descriptions.
* **Meeting & Location Setup:** * Interactive native calendar widget (`CalendarDatePicker`) to schedule sessions.
  * Location dropdown selection and static map integration.
* **Agenda / Schedule:** A vertical timeline view of upcoming study sessions and classes.
* **Chat Interface:** A placeholder messaging system complete with automatic keyboard focus (`autofocus`) for instant typing.
* **Profile:** User account management and settings interface.
